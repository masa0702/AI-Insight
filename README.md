# 調査アプリ

ブラウザ上で簡単な調査を行うためのサンプルアプリです。単一ページで動作し、質問チャットや進捗表示などの機能を備えています。

## ファイル構成

- `index.html` : 画面レイアウトと各セクションのHTMLを記述しています。
- `style.css` : 色・レイアウト・ボタンなど見た目に関するスタイルをまとめています。
- `script.js` : 画面遷移、進捗管理、チャット処理などのロジックを実装しています。

## 使い方

1. リポジトリをローカルにクローンします。
2. `index.html` をブラウザで開くとアプリが表示されます。

各ファイルの役割を把握することで、デザイン変更や機能追加を簡単に行うことができます。
